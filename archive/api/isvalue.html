_.isValue()

Checks whether the given object is a value.
Util module only.

Syntax

_.isValue(obj)

Parameters

obj
the object to test
(return value)
true if the object is a value, false otherwise.

Description

Checks whether the given object is a value. Minified defines values as all basic types (strings, booleans and numbers) and Dates.

See also..

  • _.isString() checks for a string.
  • _.isNumber() checks for a number.
  • _.isBool() checks for a boolean.
  • _.isDate() checks for a date.
