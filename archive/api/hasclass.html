.hasClass()

Checks whether at least one HTML element in the list has the given CSS class name.

Syntax

hasClass(className)

Parameters

className
the class to to find
(return value)
the first element that has the given class, or undefined if not found

Description

Checks whether at least one HTML element in the list has the given CSS class name. If yes, the first element that matches is returned. Otherwise the function returns undefined. List elements that do not have a className property will be ignored.

Example

Checks whether the element 'myElement' has the class 'myClass'. If yes, it sets the text color to red. 

if($('#myElement').hasClass('myClass'))
    $('#myElement').set('$color', 'red');
