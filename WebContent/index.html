A Truly Lightweight JavaScript Library
What?
Minified.js is a client-side JavaScript library, comparable to jQuery and MooTools in scope. Its features include DOM manipulation, animation, events, cookies and HTTP requests.
Why?

Minified is smaller!

Minified's size is about 4kB, a fraction of other libraries' size. Yet its feature set is roughly comparable to jQuery and MooTools, and even includes some additional features.

Easy to learn!

Minified has relatively few, but very powerful functions. There's no need to wade through two dozens pages filled with functions to find what you need.

No surprises!

Minified has no polyfills and does not try too hard to hide browser differences. Your JavaScript can, for better or worse, work with the same features as your HTML and CSS.
