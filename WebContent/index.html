Redefining 'lightweight' for JavaScript Libraries
What?
Minified is a client-side JavaScript library, comparable to jQuery and MooTools in scope. Its features include DOM manipulation, animation, events, cookies and HTTP requests.
Why?

Minified is smaller!

Minified is a sub-4kb JavaScript library for client-side scripting. Its feature set is comparable to jQuery and MooTools, but at 1/7th of the size.

Easy to learn!

Minified has relatively few, but very powerful functions. There's no need to wade through two dozens pages filled with functions to find what you need.

No surprises!

Minified has no polyfills and does not try too hard to hide browser differences. Your JavaScript can, for better or worse, work with the same features as your HTML.
