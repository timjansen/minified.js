MINI.parseJSON()

Parses a string containing JSON and returns the de-serialized object.

Syntax

MINI.parseJSON(text)

Parameters

text
the JSON string
return value
the resulting JavaScript object. Undefined if not valid.

Description

Parses a string containing JSON and returns the de-serialized object. If JSON.parse is defined (built-in in some browsers), it will be used; otherwise MINI's own implementation.

Example

Parsing a JSON string: 
var jsonString = "{name: 'Fruits', roles: ['apple', 'banana', 'orange']}";
var myObj = MINI.parseJSON(jsonString);
