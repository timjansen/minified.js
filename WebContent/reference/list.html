Minified Lists

Minified lists are Array-like objects provided by Minified.

Description

Minified lists are Array-like objects provided by Minified. Like a regular JavaScript array, they provide a length property and you can access their content using the index operator (a[5]). However, they do not provide the same methods as JavaScript's native array.

Minified lists are usually created using the $() function. You can also use $() to convert a JavaScript array into a Minified list, just be aware that $() will remove nulls from the lists and will flatten nested lists.

There is currently no function to convert a Minified list into a JavaScript array. The upcoming Utility module will provide one though.

The Minified Web module provides a number of helper methods for working with Minified lists:

  • collect() creates a new list using the collect function which can transform list elements or collect data from them ("map() on steriods")
  • each() iterates through all list elements
  • filter() creates a new list that contains only elements that pass the filter function's test
  • find() finds a list element or its position with the help of a find function
  • sub() creates a list that copies the elements from the specified index range

In addition to that, most methods for working with DOM and the web browser are also implemented as Minified list methods.
