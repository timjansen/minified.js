.hasClass()

Checks whether at least one element in the list has the given class name.

Syntax

hasClass(className)

Parameters

name
the class to to find
return value
the first element that has the given class, or undefined if not found

Description

Checks whether at least one element in the list has the given class name. If yes, the first element that matches is returned. Otherwise the function returns undefined.

Example

Checks whether the element 'myElement' the class 'myClass'. If yes, it sets the text color to red. 
if($('#myElement').hasClass('myClass'))
    $('#myElement').set('$color', 'red');
