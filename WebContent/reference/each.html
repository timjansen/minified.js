.each()

Invokes the given function once for each item in the list.

Syntax

each(callback)

Parameters

callback
the callback function(item, index) to invoke.
return value
the list

Description

Invokes the given function once for each item in the list. The function will be calles with the item as first parameter and the zero-based index as second.

Example

This goes through all h2 elements of the class 'section' and changes their content: 
$('h2.section').each(function(item, index) {
    item.innerHTML = 'Section ' + index + ': ' + item.innerHTML;
});
