promise.error()

Allows you to add a callback that will be called when the operation failed.

Syntax

promise.error(callback)

Parameters

callback
a function to be called when the operation has failed. The exact arguments depend on the operation. If the function returns a Promise, that promise will be evaluated to determine the state of the returned promise. If provided and it returns regularly, the returned promise will have success status. If it throws an error, the returned promise will be in the error state.
return value
a new Promises object. Its state is determined by the callback.

Description

Allows you to add a callback that will be called when the operation failed. This is a convenience function that will call then() with the only the second argument set.

Example

Simple handler for a HTTP request. 
$.request('get', '/weather.html')
    .error(function() {
       alert('Got error!');
    });
